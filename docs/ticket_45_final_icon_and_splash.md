# Ticket 45: Final SubWatch Icon and Splash

## Final icon direction
- Warm dark rounded-square base.
- One dark subscription card object.
- One integrated amber pulse/watch mark.
- Minimal cream signal lines for readability at small sizes.
- No text, no rupee, no wallet, no chart, no destructive metaphor.

## Splash direction
- Warm dark background using the same brand base color as the icon.
- Transparent launch mark centered on the native Android splash.
- No long intro or theatrical motion.
- Flutter loading handoff uses the same mark with a short scale/fade entrance.

## Wiring summary
- Legacy launcher PNGs updated across all Android mipmap densities.
- Adaptive icon resources added for Android 8+.
- Round icon attribute added in the manifest.
- Native launch background updated to use the transparent launch mark.
- Post-splash loading state aligned to the same mark.
